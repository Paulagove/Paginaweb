¿Qué es el lenguaje HTML?

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y diseñar páginas web. Se trata de un lenguaje de marcado que define la estructura y el contenido de una página web mediante etiquetas o elementos. Estos elementos indican al navegador cómo mostrar el contenido, desde encabezados, párrafos y listas hasta imágenes, videos, enlaces y formularios. La evolución de HTML ha traído consigo varias versiones, siendo HTML5 la más reciente. HTML5 introdujo nuevas etiquetas, atributos y funcionalidades que permiten una web más dinámica, accesible y rica en contenido multimedia. Además, HTML se combina frecuentemente con CSS (Cascading Style Sheets) para controlar la presentación y el diseño, y con JavaScript para añadir interactividad y funcionalidad a las páginas web.
